📌 핵심 변경 원인 및 기술 가이드

1. Sizzle 엔진 제거 (Native querySelectorAll 전환):
jQuery 4.0.0부터 비표준 선택자(:first, :last, :eq(), :visible, :hidden, :radio, :checkbox, :header)를 탐색 구문 내에서 직접 처리하던 Sizzle이 제거되었습니다. $()나 find()에 비표준 문자열이 들어갈 경우 브라우저 DOMException: Failed to execute 'querySelectorAll' 예외가 발생하므로, Pure API 메서드(.first(), .last(), .filter()) 또는 표준 CSS 속성 선택자로 분리하였습니다.

2. jQuery.cssNumber 객체 삭제 대처:
jQuery 4.0.0 Core에서 jQuery.cssNumber가 제거되어 jquery.transit 서드파티 라이브러리 초기화 중 Cannot set properties of undefined (setting 'scale') 에러가 발생하던 문제를 객체 자동 초기화 방어 코드(t.cssNumber = t.cssNumber || {})로 해결하였습니다.

3. jQuery.fn.sort 별칭 삭제 대처 (.get().sort() 전환):
jQuery 4.0.0에서 jQuery.fn.sort 메서드가 삭제되었습니다. .map() 결과 컬렉션에 직접 .sort()를 호출 시 TypeError: ...map(...).sort is not a function이 발생하므로 .map(...).get().sort(...)로 순수 JS Array 변환 후 정렬하도록 개작했습니다.

4. Null / Undefined 런타임 참조 방어:
DOM 렌더링 미완료 및 Flickity 미초기화 시 발생하는 참조 오류를 예방하기 위해 널 체킹(gnbEl ? gnbEl.clientHeight : 0, if (flkty && flkty.slides))을 적용했습니다.
